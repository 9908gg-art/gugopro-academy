從零開始的策略交易學習之旅

歡迎來到 GugoPro 財經教學學院。在這裡，我們依據「學程晉級制」規劃了四個學習階段，從最基礎的觀念心態到股票、期貨實務，最後進入高階的統計套利與量化交易，為您建立全面且系統性的交易邏輯。

學院學習地圖

請點選您目前的學習進度，進入對應的系統化學程單元：

Level 1: 🌟 啟蒙階段 3 個科目

交易啟蒙與心態資金管理

探索市場運作的本質。從股票與期貨的定義開始，學習保證金交易風險、停損紀律及克服恐懼與貪婪的心態鍛鍊，奠定最關鍵的安全邊際基礎。

  • 1.1 交易基本常識 (Market Basics)
  • 1.2 交易心態與三大禁忌
  • 1.3 找出支撐與壓力：進出場點如何找出風報比？
進入學程
Level 2: 📈 股票學程 3 個科目

股票策略與存股價值投資

專注於股權投資的核心策略。學習葛拉漢價值投資的安全邊際、財報分析要領，以及如何篩選高股息標的進行長線存股，並輔以技術分析尋找波段進出場點。

  • 2.1 價值投資大師心法
  • 2.2 高殖利率存股策略
  • 2.3 技術分析與波動交易
進入學程
Level 3: 🌪️ 期貨學程 3 個科目

期貨價差與跨期對沖避險

深入期貨市場的微觀機制。從保證金槓桿計算，學習多空方向操作、合約乘數換算，進而掌握跨期價差套利（Calendar Spread）與如何利用衍生性商品進行避險。

  • 3.1 期貨合約與保證金運作
  • 3.2 價差與跨期套利實務
  • 3.3 跨市場對沖與風險平衡
進入學程
Level 4: ⚖️ 量化學程 3 個科目

量化交易與統計套利實戰

踏入最前沿的系統化交易殿堂。利用統計學協整性測試（Cointegration）篩選配對標的，透過 Z-Score 產生均值回歸交易信號，並運用 Python 進行歷史數據回測。

  • 4.1 配對交易實戰 (Pairs Trading)
  • 4.2 Pine Script 程式入門
  • 4.3 策略回測與跟單實務
  • 4.4 傳奇大師交易心法應用
進入學程
