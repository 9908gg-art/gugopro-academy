Unit 1.3

找出支撐与壓力：进出场点如何找出风報比？

在进入金融市场交易前，您必須看清市场的「地板」与「天花板」。本单元将用最淺顯易懂的白話圖解，帶您認識支撐与壓力，並学會如何利用極窄的停损鎖定高倍數的风险報酬比。

1. 什麼是支撐与壓力？（白話的地板与天花板法则）

在看股票或期货的 K 線圖时，您會發現价格有时跌到某個位置就跌不動了，有时漲到某個位置就漲不上去了。這就是市场的 **「支撐」** 与 **「壓力」**：

  • 支撐位 (Support) ➡️ 市场的「橡皮彈性地板」
    想像价格是一個掉到地上的橡皮球。當它跌到某個价格区间时，會有一股力量把它向上彈起。
    為什麼會這樣？ 因為此时無數的买家（包括大资金主力、散戶、法人）都一致公認這個价格「非常便宜」，因此紛紛掛单买进。當「买盤」遠大於「賣盤」时，价格就會止跌回升。
  • 壓力位 (Resistance) ➡️ 市场的「防護罩天花板」
    想像价格是一個往上拋的皮球，撞到天花板後會反彈跌下來。
    為什麼會這樣？ 因為在這個价格区间，之前买入被套牢的人急著保本賣出，而低点买入的人想獲利了结。當「賣盤」遠大於「买盤」时，价格就很難突破。
為什麼要找出它们？
因為支撐与壓力是全市场交易者共同達成的「共識边界」。找出地板（支撐），是為了找到安全、不容易賠大錢的买进点；找出天花板（壓力），是為了找到合理的利潤离场目标。

低风险、高報酬的黄金进场时機示意圖

※ 示意圖：當我们在支撐位上方緊鄰處进场，我们承擔的风险極小（Risk = 1），但向上的獲利空间極大（Reward = 7）。這就是典型的「低风险高報酬」时機。

2. 如何找出完美的进场点、停损点与出场点？

一旦在圖表上識別出地板（支撐）与天花板（壓力），我们就能精確判定交易的黄金三元素：

  1. 进场点 (Entry) ➡️ 貼近地板的时機
    最佳的买入时機，必須选在**剛好回档到支撐位（地板）上方 1% ~ 2% 處**。因為這时候买入，你离下方的停损防線距离最近。如果价格已经漲到半空中，就不應該进场，因為這属於「高风险、低報酬」的追高行為。
  2. 停损点 (Stop Loss) ➡️ 跌穿地板的防線
    将停损点设定在**支撐位下方一点点**。一旦价格真的有效跌穿地板，代表支撐失效、您的看多邏輯被市场否定。因為进场点貼得極近，您的虧损幅度會非常微小。
  3. 出场点 (Take Profit / 停利) ➡️ 撞擊天花板前的落袋
    将目标出场点设定在**壓力位（天花板）下方一点点**。在价格再次撞牆反彈回跌前，順利獲利平倉落袋為安。

3. 贏家密碼：如何只看圖表百分比算出「风報比」？

要计算一筆交易值不值得做，根本不需要繁瑣地计算您的總资金。您只需要**直接看商品的圖表价格与百分比距离**即可：

风险距离 % = 进场价格 到 支撐停损价格 的下跌百分比距离
報酬距离 % = 进场价格 到 壓力停利价格 的上漲百分比距离
风险報酬比 (风報比) = 风险距离 % : 報酬距离 %

💡 实战看圖估算步驟：

  • 第一步：找出防守地板（支撐）与獲利天花板（壓力）
    股票 A 目前价格為 $100 元。您在圖表上看出強大支撐地板在 $98 元，上方壓力天花板在 $108 元。
  • 第二步：计算百分比距离
    风险距离 % = $(100 - 98) / 100 = 2\%$ （跌破 2% 就停损）。
    報酬距离 % = $(108 - 100) / 100 = 8\%$ （上漲 8% 就停利）。
  • 第三步：得出风险報酬比
    這筆交易的风報比為 **2% : 8%**，簡化後就是 **1 : 4**（冒著 1 倍的风险去賺取 4 倍的利潤）。

在实战中，我们應該只參与风報比在 1:3 以上（如 1:4、1:5）的划算交易。

4. 黄金进场法则：只在「窄停损、寬獲利」的时段出手

為什麼我们強調「进场的时機点（Timing）」非常重要？因為即使是同一個商品，在不同位置进场，风報比會天差地別：

  • 完美时機（剛回档到支撐地板上方）
    如前所述，您买在 $100 元，停损设 $98 元（风险 2%），目标 $108 元（報酬 8%）。风報比 = 1:4。這是一筆極具价值的交易。
  • 錯誤时機（价格已上漲，忍不住在半空追高）
    如果股票 A 已经漲到了 $105 元，此时您忍不住追高买入：
    您的停损点依然必須设在支撐地板 $98 元（此时风险距离拉大到 **6.6%**）。
    您的目标停利点依然在天花板 $108 元（此时報酬距离縮小到只剩 **2.8%**）。
    结果：风報比變成了 **6.6% : 2.8%**（约為 2.3 : 1，意即冒著賠 2.3 元的风险去賺 1 元）。這是一筆極度愚蠢、完全不划算的交易，即使看对也容易賠大錢。

结论：只在进场点極度接近支撐線（地板）时出手。如果价格已经起飛、遠离支撐，最佳的选择是放棄這筆交易，耐心等待下一次拉回的機會。
