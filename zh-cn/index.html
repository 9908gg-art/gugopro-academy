学程晉級制 ‧ 系统化交易学習路徑

從零开始的策略交易学習之旅

欢迎來到 GugoPro 财经教学学院。在這裡，我们依據「学程晉級制」规划了四個学習阶段，從最基礎的觀念心态到股票、期货实務，最後进入高阶的统计套利与量化交易，為您建立全面且系统性的交易邏輯。

学院学習地圖

请点选您目前的学習进度，进入对應的系统化学程单元：

Level 1: 🌟 啟蒙阶段 3 個科目

交易啟蒙与心态资金管理

探索市场運作的本质。從股票与期货的定義开始，学習保证金交易风险、停损紀律及克服恐懼与貪婪的心态鍛鍊，奠定最關鍵的安全边際基礎。

  • 1.1 交易基本常識 (Market Basics)
  • 1.2 交易心态与三大禁忌
  • 1.3 风险控制与资金管理
进入学程
Level 2: 📈 股票学程 3 個科目

股票策略与存股价值投资

专注於股權投资的核心策略。学習葛拉漢价值投资的安全边際、财報分析要領，以及如何篩选高股息标的进行長線存股，並輔以技术分析寻找波段进出场点。

  • 2.1 价值投资大师心法
  • 2.2 高殖利率存股策略
  • 2.3 技术分析与波動交易
进入学程
Level 3: 🌪️ 期货学程 3 個科目

期货价差与跨期对沖避险

深入期货市场的微觀機制。從保证金杠杆计算，学習多空方向操作、合约乘數換算，进而掌握跨期价差套利（Calendar Spread）与如何利用衍生性商品进行避险。

  • 3.1 期货合约与保证金運作
  • 3.2 价差与跨期套利实務
  • 3.3 跨市场对沖与风险平衡
进入学程
Level 4: ⚖️ 量化学程 3 個科目

量化交易与统计套利实战

踏入最前沿的系统化交易殿堂。利用统计学協整性測試（Cointegration）篩选配对标的，透过 Z-Score 产生均值回归交易信号，並運用 Python 进行歷史數據回測。

  • 4.1 配对交易实战 (Pairs Trading)
  • 4.2 Python 量化交易入门
  • 4.3 传奇大师交易心法應用
进入学程
