量化交易学程 (Quantitative Trading)

欢迎來到 Level 4 学程。本单元是学院的核心精華，专注於利用數学与统计学建立市场中性策略，並利用 Python 进行资料分析与策略回測，最終学習大师的资金控管心法。

返回学院地圖
单元 4.1 5 個章节

配对交易实战 (Pairs Trading)

深入探讨统计套利与均值回归原理，学習協整性測試、Z-Score 价差信号计算，以及配对交易的建倉、平倉与风险控管原则。

进入学習
单元 4.2 规划中

Python 量化交易入门

学習使用 Python 抓取金融歷史數據、计算技术指标与统计量，並建立自動化策略回測框架，用數據驗证交易策略的有效性。
