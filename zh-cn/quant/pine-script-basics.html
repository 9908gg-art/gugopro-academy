Unit 4.2

TradingView Pine Script 程式交易入门与指标开發

学習如何使用 TradingView 专属的 Pine Script 腳本語言，編寫属於您的技术指标、自動化划線，並為未來的程式化自動交易打下穩固基礎。

1. 什麼是 Pine Script？

Pine Script 是由 TradingView 官方开發的輕量級腳本語言。它的語法非常簡单好学，類似於 Python，但針对「金融圖表」与「技术指标」做了極致的優化。

以往在 Python 中需要寫上百行代碼才能載入歷史 K 線、计算均線並畫圖；但在 Pine Script 中，您只需要不到 5 行代碼，就能完美计算並繪製出一條自訂技术指标！

2. Pine Script 的兩大運行模式

在开始寫程式前，您必須区分 Pine Script 的兩種主要程式结构：

  • 指标 (indicator)：用於在圖表上繪製視覺輔助工具。例如自訂的 MACD、多重均線通道或我们在配对交易中使用的 Z-Score 价差線。
  • 策略 (strategy)：除了畫圖，還包含了「买入 (Buy)」与「賣出 (Sell)」的邏輯，可用於在歷史數據上自動執行交易模擬並跑出績效報表。

3. 实战：編寫您的第一個 Pine 均線策略

我们以最经典的「黄金交叉与死亡交叉」均線策略為例。當 5 日均線往上突破 20 日均線时做多，往下突破时做空。以下是完整的 Pine Script 原始碼：

//@version=5
strategy("GugoPro 均線交叉战术", overlay=true)

// 1. 设定輸入參數
fastLength = input.int(5, title="快線天數")
slowLength = input.int(20, title="慢線天數")

// 2. 计算移動平均線 (SMA)
fastMA = ta.sma(close, fastLength)
slowMA = ta.sma(close, slowLength)

// 3. 繪製均線到圖表上
plot(fastMA, color=color.aqua, title="快線")
plot(slowMA, color=color.orange, title="慢線")

// 4. 定義进出场交叉讯号
longCondition = ta.crossover(fastMA, slowMA)
shortCondition = ta.crossunder(fastMA, slowMA)

// 5. 執行策略模擬
if (longCondition)
    strategy.entry("做多", strategy.long)

if (shortCondition)
    strategy.close("做多")

4. 如何在 TradingView 中執行這段程式？

  1. 打开您的 **[TradingView 圖表页面](https://www.tradingview.com/?aff_id=168714)**。
  2. 点选底部的 **「Pine 編輯器 (Pine Editor)」** 页籤。
  3. 清除所有默認文字，将上方那段程式碼**複製並貼上**进去。
  4. 点擊右上角的 **「儲存 (Save)」**，接著点擊 **「添加到圖表 (Add to Chart)」**。
  5. 此时您的圖表上就會繪製出快慢均線，並且在底下會多出 **「策略測試器 (Strategy Tester)」** 看板，展示這套策略在过去歷史中的盈虧成效！
